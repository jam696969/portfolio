# AJOL Portfolio 🎮

A retro arcade-themed portfolio website for **Al John Oga Lomocso** — Full Stack Web Developer.

## ✨ Features

- **Arcade/Gaming UI** — Neon colors, pixel fonts, CRT scanlines, retro aesthetics
- **AI Chatbot Assistant** — Ask about skills, projects, experience, and background
- **Light/Dark Mode** — Toggle with persistence via localStorage
- **3D Animated Background** — Floating geometric shapes using Three.js with mouse parallax
- **PDF Resume Download** — Direct download with native browser dialog
- **Responsive Design** — Optimized for desktop, tablet, and mobile
- **Scroll Animations** — AOS (Animate On Scroll) library integration

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure & semantics |
| CSS3 | Styling, variables, animations |
| JavaScript (Vanilla) | DOM manipulation, interactivity |
| Three.js | 3D background scene |
| AOS | Scroll-triggered animations |
| Font Awesome | Icons |
| Google Fonts (Press Start 2P, VT323) | Typography |

## 🚀 Deployment

This is a fully static site. Deploy to GitHub Pages:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/jam696969/portfolio.git
git push -u origin main
```

Then enable Pages: Repo → Settings → Pages → Branch: `main`, folder: `/ (root)`
